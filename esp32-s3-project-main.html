This project documents the complete design and layout of a custom ESP32-S3 development module, completed as an independent engineering project during second year Electrical and Electronics Engineering at the University of Western Australia.

The board was designed from schematic capture through to manufacturing-ready Gerber files using KiCad 9, targeting JLCPCB 4-layer manufacturing with the JLC04161H-3313 stackup. All design rules were verified against JLCPCB standard capabilities and the IPC-2141 impedance standard using JLCPCB's stackup parameters.

Key features include native USB-C via the ESP32-S3 built-in USB Serial/JTAG controller, an ENS210 temperature and humidity sensor over I²C, a WS2812B addressable RGB LED, TLV75801 800mA LDO with thermal package selection, full GPIO breakout headers, and hardware BOOT and RESET buttons.

Design Process

Phase 1 — Schematic Capture
Component selection and schematic design in KiCad 9
Selected components referencing Espressif application notes and IC datasheets. Designed full schematic including USB-C with CC pull-downs, TLV75801 LDO, ENS210 I²C sensor, WS2812B RGB LED, and GPIO breakout. ERC verified with 0 errors.
Full schematic — KiCad 9 schematic editor
Phase 2 — Impedance Calculation
IPC-2141 differential pair impedance for JLCPCB JLC04161H-3313 stackup
Used JLCPCB stackup JLC04161H-3313: PP prepreg 3313×1 with h = 0.0994 mm and Dk = 4.4. Applied IPC-2141 edge-coupled microstrip formula to determine trace width of 0.18 mm at 0.15 mm spacing achieves 90 Ω differential impedance — within USB 2.0 spec of 90 Ω ±15%. Note: values updated from original PCBWay calculation (h = 0.1855 mm, Dk = 4.74, w = 0.2251 mm).

Strategic placement following signal flow and thermal considerations
Placed components following signal flow from USB-C through ESD protection, LDO, and ESP32-S3 module. Decoupling capacitors placed immediately adjacent to IC power pins. BOOT and RESET buttons at board edge for accessibility. (Fig. 1 — see left) (Fig. 1 — see left)

Impedance-controlled routing, differential pairs, and GND pours on all 4 layers

Key Engineering Decisions

Native USB over USB-to-UART bridge
Uses the ESP32-S3 built-in USB Serial/JTAG controller via GPIO19 (D−) and GPIO20 (D+) instead of a CP2102N bridge. Eliminates one IC and reduces BOM cost without sacrificing programming or debug capability.
Impedance-controlled USB D± — recalculated for JLCPCB JLC04161H-3313
Originally calculated for PCBWay stackup (h = 0.1855 mm, Dk = 4.74, w = 0.2251 mm). Recalculated for JLCPCB 3313 stackup: h = 0.0994 mm, Dk = 4.4, giving trace width 0.18 mm at 0.15 mm spacing for 90 Ω differential — within USB 2.0 spec of 90 Ω ±15%.
GPIO0 excluded from header — strapping pin protection
GPIO0 determines boot mode. Reserved exclusively for the BOOT button to prevent external circuitry accidentally pulling it low. GPIO45 and GPIO46 are exposed but flagged as strapping pins on the silkscreen and pinout reference.
4-layer stackup with dedicated GND plane on L2
Provides continuous GND reference directly beneath the signal layer, enabling impedance control for USB D±, reducing EMI from the ESP32-S3 Wi-Fi/BT radio, and providing a low-inductance return path for high-frequency signals.
LDO package selection based on thermal analysis
TLV75801 WSON-6 selected over SOT-23: RθJA = 80.3°C/W vs 176.9°C/W. At 300mA load (P = 0.51W), WSON-6 gives only +41°C junction rise vs +90°C for SOT-23. Thermal pad soldered to GND copper pour.
Battery voltage divider — ADC impedance flagged for Rev 2
R6/R7 (1MΩ/200kΩ) gives correct 1:6 ratio but ADC source impedance ~167kΩ exceeds ESP32 recommended 100kΩ. Rev 2 will use 100kΩ / 20kΩ maintaining ratio at 16.7kΩ source impedance.

Future Improvements — Rev 2

Battery voltage divider impedance correction
Replace R6/R7 with 100kΩ / 20kΩ to reduce ADC source impedance from 167kΩ to 16.7kΩ, within the ESP32 recommended 100kΩ maximum. Optionally add GPIO-controlled enable to eliminate quiescent drain during sleep.
WS2812B power supply clarification
Confirm WS2812B operation on 3.3V rail. If 5V operation preferred, add a single-transistor level shifter on the DIN line to ensure logic high exceeds 0.7×VDD input threshold.
Footprint library consolidation
Migrate all custom footprints (ESP32-S3-MINI-1-N8, USB4105) into a project-local KiCad library to eliminate the footprint mismatch warnings in DRC and ensure reproducibility.
Consider ESP32-S3-MINI-1-N8R8 (PSRAM variant)
Same pinout and footprint as current module but adds 8MB PSRAM — no PCB changes required. Beneficial for display framebuffers, audio buffers, or any memory-intensive application.
